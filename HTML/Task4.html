CCA-Certification Course in Computer Application

What is CCA certificate?

CCA,an acronym for Certificate in Computer Application,is a vacational certification program that aims to upgrade an individual's
computer skills,enhancing the employability

What is qualification for CCA?

Candidates must have passed graduation or equivalent level(with mathematics as a subject of study at 10+2 or
graduation level)examination from a recognized university.He/She must have obtained at least 50%(relaxable for
reserved category candidates)the marks at graduation or equivalent level

Which subject is CCA

Certificate in Computer Application is a certificate course to study computer application in depth.The course impacts theoratical,practical and technical knowledge to
its learners about various computer tools that are used in day to day life

C++ -Object Oriented Programming using C++

What is C++?

C++(said C plus plus) is an object-oriented computer language created by notable computer scientist created
by notable computer scientist by Bjarne Stroustrop as a part of the evolution of the C family of languages.

What is difference between C and C++?

C is a structural or procedural programming language that was used for system application and low-
level programming language applications.Whereas C++ is an object-oriented programming language having some
additional features like Encapsulation,Data Hiding,Data Abstraction,Inheritance,Polymorphism,etc

What is the Oops concept?

Object-oriented programming(OOP) is a computer programming model that organizes software design
around data,or objects,rather than function and logic.An object can be defined as a data field that has
unique attributes and behaviour
